# Cleaner AI - 智慧圖片清理工具 v29.0

**Cleaner AI** 是一款強大的本地端圖片清理工具，專為攝影師、設計師與資料整理者設計。結合了傳統的 **pHash (感知雜湊)** 演算法與先進的 **AI 深度學習模型 (EfficientNet-B3)**，能精準識別重複圖片與相似構圖，並具備自動損毀檔案偵測功能。

## ✨ 主要功能

* **🚀 雙重掃描引擎**：
    * **pHash 模式**：極速比對，適合找出完全相同的圖片。
    * **AI 模式**：使用 EfficientNet-B3 提取特徵向量，能識別「構圖相似」、「濾鏡不同」或「輕微裁切」的圖片。
    * **智慧掃描 (推薦)**：先用 pHash 快速過濾，再用 AI 進行深度分析，兼顧速度與準確度。
* **🧠 智慧閥值重置**：掃描途中若調整相似度閥值，程式會自動判斷並重新掃描，確保結果準確。
* **⚡ 多執行緒與並行運算**：充分利用 CPU 多核心進行運算，操作介面流暢不卡頓。
* **🛡️ 安全刪除機制**：
    * 支援刪除至資源回收桶 (可復原)。
    * 可選擇保留原圖或同時刪除。
    * 自動偵測並標記無法讀取的損毀檔案。
* **💻 跨平台支援**：優化了 Windows、macOS 與 Linux 的檔案開啟與瀏覽體驗。
* **☁️ 線上更新檢查**：自動檢查 GitHub 最新發布版本，確保您使用的是最新版。

## 📥 下載與安裝

1. 前往本頁面的 **[Releases (發行版本)](../../releases/latest)** 頁面。
2. 下載最新版本的 `CleanerAI_v29.zip`。
3. **解壓縮** 該檔案到您喜歡的位置。
4. 進入資料夾，雙擊 `CleanerAI_v29.exe` 即可啟動。

## 📖 使用說明

### 步驟 1：啟動程式
雙擊 `CleanerAI_v29.exe` 啟動。首次啟動可能需要幾秒鐘載入 AI 模型。

### 步驟 2：選擇目標
點擊左側的 **「1. 選擇資料夾」** 按鈕，選取您想要檢查重複圖片的目錄。

### 步驟 3：設定掃描模式 (可選)
* **智慧掃描 (Smart)**：推薦選項。速度快且精準。
* **純 AI 模式**：速度較慢，但能找出「雖然修改過但構圖相似」的圖片。
* **相似度門檻**：透過滑桿調整（預設 95%）。
    * 如果掃描途中調整滑桿，程式會自動重新掃描以符合新標準。

### 步驟 4：執行掃描
點擊 **「2. 開始掃描」**。
* 程式會顯示當前進度（階段 1：雜湊比對 / 階段 2：AI 分析）。
* 掃描到的重複圖片會即時顯示在右側列表。

### 步驟 5：清理結果
* **檢視圖片**：點擊列表中的 `🔍` 可開啟圖片，點擊 `📂` 可開啟所在資料夾。
* **選取刪除**：
    * 勾選您想要刪除的項目（通常是標示為 `Duplicate` 的圖片）。
    * 使用 **「✨ 全選當前列表」** 可快速選取所有重複項目。
    * 點擊 **「3. 刪除勾選項目」** 執行刪除（檔案將被移至資源回收桶）。
* **損毀檔案**：若有發現無法讀取的圖片，會出現在「損毀檔案」分頁中，可一併清理。
